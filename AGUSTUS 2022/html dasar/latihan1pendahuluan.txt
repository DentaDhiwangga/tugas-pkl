# HTML-Dasar
Mempelajari HTML Dasar Materi 1

<HTML> Merupakan Singkatan Dari Hypertext Markup Languange dan bukan bahasa pemrograman, 
HTML disebut Hypertext Markup Languange karena didalamnya terdapat serangkaian Markup. 
Fungsi Markup adalah berfungsi untuk memberitahu broswer bagaimana konten akan ditampilkan.
HTML Pertama kali diciptakan oleh Tim Berners-Lee pada akhir 1991. Selain HTML Tim Berners-Lee 
juga menemukan:
-HTTP
-HTML
-WWW
-WEB 
-BROSWER
-WEB SERVER
-WEB PAGE.

Tim Berners-Lee Sekarang juga menjadi direktur organisasi yang bernama W3C (WORD WIDE WEB CONSORTIUM).
World Wide Web Consortium (W3C) adalah komunitas internasional di mana anggota organisasi ,
staf penuh waktu , dan masyarakat bekerja sama untuk mengembangkan standar Web . Dipimpin oleh penemu
Web dan Direktur Tim Berners-Lee dan CEO Jeffrey Jaffe , misi W3C adalah memimpin Web ke potensi penuhnya.

Ringkasan:
HTML untuk mengelola serangkaian data dan informasi sehingga suatu dokumen dapat diakses dan ditampilkan
di Internet melalui layanan web.
CSS untuk mendesain, membentuk, serta mengubah tampilan halaman sebuah website. CSS dapat bekerja dan
berlaku melalui tag HTML.
